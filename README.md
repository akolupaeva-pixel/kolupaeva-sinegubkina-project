# kolupaeva-sinegubkina-project

# Идея
Исследование региональных особенностей речи: грамматический, лексический и синтаксический аспекты. Изучение того, насколько сильно региональные отличия проявляются в разных видах текстов: публицистика, художественная проза, посты в соцсетях, видео на ютубе.

Источники: Национальный корпус русского языка, региональные СМИ, Youtube. 
## Состав группы

Милана Синегубкина (github: milana13mnmn) и Анастасия Колупаева (github: akolupaeva-pixel)

## Разделение обязанностей

- Милана Синегубкина: проверка гипотез о лексических аспектах текстов, варьирующихся в зависимости от региона
- Анастасия Колупаева: проверка гипотез о синтаксических особенностях текстов в зависимости от региона, сравнение отличий в разных типах текстов
